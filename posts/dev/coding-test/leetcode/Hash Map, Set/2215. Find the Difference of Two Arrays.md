the key is category because they create this problem to solve with hash

```kotlin
class Solution {
    fun findDifference(nums1: IntArray, nums2: IntArray): List<List<Int>> {
        // number 1: setting hash
        val set1 = nums1.toSet()
        val set2 = nums2.toSet()

        // number 2: subtract and list
        val diff1 = set1.subtract(set2).toList()
        val diff2 = set2.subtract(set1).toList()

        // number 3: return
        return listOf(diff1, diff2)
    }
}
```
