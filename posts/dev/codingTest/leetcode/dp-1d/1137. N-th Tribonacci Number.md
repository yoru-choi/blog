이런흐름으로 생각해서 만들었는데 gpt와 별다른 차이가 없이 만들었다
오랜만에 eazy 문제를 푸니까 선녀가 따로없다

트리보나치 수열 간단하다

이전의 항 3개의 합이 현재항의 정답이다 즉 아래와같이 3개를 더해주고 그값만 리턴하면된다

```kotlin

class Solution {
    fun tribonacci(n: Int): Int {
        if (n == 0) return 0
        if (n == 1 || n == 2) return 1

        val dp = IntArray(n + 1)
        dp[0] = 0
        dp[1] = 1
        dp[2] = 1

        for (i in 3..n) {
            dp[i] = dp[i - 1] + dp[i - 2] + dp[i - 3]
        }

        return dp[n]
    }
}
```
