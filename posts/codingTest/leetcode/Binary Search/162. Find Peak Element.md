풀이 방식을 보고 전혀 모르겠다
그냥 배열로 풀면 풀어지지만 binery search 방식은 이렇다고 한다

```kotlin

class Solution {
    fun successfulPairs(spells: IntArray, potions: IntArray, success: Long): IntArray {
        potions.sort()
        val n: Int = spells.size; val m: Int = potions.size
        val res: IntArray = IntArray(n)

        for(i in 0 until n) {
            var left: Int = 0; var right: Int = m
            while(left < right) {
                var mid: Int = left + (right-left)/2
                var currValue: Long = spells[i].toLong() * potions[mid].toLong()
                if (currValue >= success) {
                    right = mid
                } else {
                    left = mid+1
                }
            }
            res[i] = m-left
        }

        return res
    }
}
```
