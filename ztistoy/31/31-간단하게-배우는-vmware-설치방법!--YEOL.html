- vmware란 

 

Windows 또는 Linux PC에서 단일 가상 머신을 실행하기에 적합한 유틸리티입니다.

쉽게 말하자면 vmware라는 프로그램을 사용하면 윈도란 운영체제 내에 

다른 운영체제 더 사용할 수 있게 해 줍니다. 

https://www.vmware.com/kr.html  

위의 경로는 홈페이지 사이트이고 아래는 다운로드하기 위한 화면 예시입니다.

https://www.vmware.com/kr/products/workstation-player/workstation-player-evaluation.html

위와 같은 다운로드 url에서 다운로드 가능합니다.

 

 

저의 컴퓨터 os는 윈도기 때문에 윈도 기반으로 사용합니다.

 

 

다운로드한 파일로 install 진행해주시면 됩니다.

 

enhanced keyboard driver 키보드 입력을 안정적으로 사용해겠냐는건데 큰 차이 안 나서 

체크 안 해주셔도 됩니다. 궁금하시면 체크하세요 ㅋㅋ

 

용량에 여유가 있다면 C드라이브로 진행해줍니다.

나머지는 전부 next를 클릭하여 진행합니다. 별다른 거 없이 next만 눌러주셔도 됩니다.

 

자 vmware 다운로드가 완료되었습니다.

너무 간단하게도 실행된 vmware를 실행해줍니다.
