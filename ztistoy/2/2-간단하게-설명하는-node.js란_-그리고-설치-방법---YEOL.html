Node.js 란

node.js 는 웹브라우저에 종속적인 javascript를 V8엔진을 통해 실행할 수 있는 환경(런타임)을 제공해줍니다.

-V8엔진 역할:  javascript를 node.js가 v8엔진에게 전달해서 기계어로 번역합니다. javascript는 사람을 위한 프로그래밍 언어이며 컴퓨터는 기계어로 번역해주지 않으면 이해하지 못합니다. 

 

설치 방법

 

node.js를 직접 검색해서 아래와 같은 링크를 타고 들어가면됩니다.

https://nodejs.org/en/

 

node.js를 설치하면 npm도 같이 설치됩니다.

npm 이란 node package manager의 약어로 node.js로 작성된 패키지들을 관리하는 프로그램입니다.

 

공부 목적이시라면 최신 버전을 다운로드해주시고 실무 목적이라면 안정적인(stable) 버전을 다운로드해주세요.

 

 

다운로드한 파일을 설치하시고 쭉 진행해주시면 됩니다. 

환경변수를 설정하고 간단하게 npm -v 나 node -v를 사용해서 설치된 프로그램의 버전을 확인해보세요.

(자동으로 환경변수 설치가 안되어있다면 https://yeolceo.tistory.com/3 )
