궁극적으로 보면 제가 한 일들을 정리하고 글로 만들기 때문에 기술적으로 도움이 되지만

기술적인 부분 때문에 시작하진 않았습니다. 에드센스를 붙여서 하루에 많으면 3원도 법니다.

 

 

-기재하려고 스터디 한 부분

Aws -> EC2 사용, aws DB 만들기, aws 보안 그룹 만들어서 db와 ec2 연동

kafka -> kafka 클러스트링 후 node.js 에서 큐 메시지 사용

java -> Spring MVC 패턴 사용한 개발 with gradle 

python -> 플라스크 사용해서 restApi 서버 만듦.

 

-스터디할 부분

english -> 웬만하면 비는 시간엔 영어를 하니까 아래 리스트 녀석들이 뒤로 밀리는 편(크리스 영어 개꿀)

typescript -> 진행 중이고 다음 프로젝트 때 node.js 서버 구현 시 타입 스크립트 적용할 예정입니다.

nosql -> mongodb 테스트 프로젝트는 진행함. 나중에 typescript 쓰면서 지정된 db를 사용하지 않는 프로젝트할 때 사용해볼 예정

linux -> 칼리 리눅스를 통한 기초 해킹 구현 및 초급 어셈블리어 사용(조금씩 보는 중)

아파치 드루이드 ->  파이썬 서버에서 구동해서 실행되는 파일도 받았는데 눈이 안감.

 

 

-초보자 대상 Java(spring), Js(node), Python(django) 이런 종류의 오프라인 강의를 하게된다면

생각하는 흐름 - 5~6주 동안 주 2~4시간 진행합니다.

인원을 모음 및 공지사항

1. java 학습을 위한 노트북 지참

2. 강의비 10만 원 (완강 시 전액 반환하고 한주 안 나올 때마다 2 만씩 차감 스터디실 대여비는 당연히 내가냄)

3. 모집 인원 약 3~5명 , 너무 많아지면 1차 면접 진행 (왜냐 나도 가르치는데 진지하니까 배울 사람도 진지했으면 함

ㄹㅇ 실무 전투를 위한 팁 같은 것도 잘 알려주는 편임)

 

지금 만든 주차별 교육(변동 가능성 있으며 학생들 실력에 따라 더 디테일하게 진행 가능)

(만약 전체적으로 다들 좀 해봤으면 핵심위주로 짚고 넘어갈 듯?)
