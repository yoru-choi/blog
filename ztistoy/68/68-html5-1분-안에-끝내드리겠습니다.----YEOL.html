-html 이란 

 

웹페이지를 표현하기 위한 프로그래밍 언어입니다. 

 

-약자를 해석해보자면-

-> HyperText

링크를 걸 수 있는 텍스트를 말합니다.  yeolceo.tistory.com 이런 느낌의 링크 글 말이죠!

(제 블로그 링크는 훼이크... )

 

-> Mark up 

<div> 문장 </div> 이런 식으로 표시 위에 글자를 얹습니다. 이게 마크업 언어입니다.

태그를 이용해서 문서 위에 데이터를 표현하는 언어를 말합니다.

 

-> Language

위에 두 가지 언어를 합친 언어라는 표현을 하기 위해 마지막에  language를 붙여줍니다.

 

말하자면!

링크를 걸 수 있는고 아래처럼 마크업으로 표시하는 프로그래밍 언어입니다.

 

<html>
    <head>
	   document에 보이는 부분 이외의 정보를 처리합니다.
    </head>
    <body>
    
       <a href="https://yeolceo.tistory.com/">이렇게 링크 걸어서 페이지를
       이동할수있기 때문에 하이퍼 텍스트입니다.</a>
       
		document에 내용이 보이는 부분입니다.
       
    </body>
</html>

 

 


감싸 놓은 라인 내부가 document 부분입니다.

 

 

 

html 설명 벌서 끝났습니다!!!

 

그런데 말입니다... 

 

html 이 빠르게 발전하면서 브라우저마다 html을 표현하는 방식이 달랐고 이것이 문제가 되었습니다.

그래서 html 표준이 필요했고 html 5라는 표준안이 만들어진 것입니다.

 

 

 

-html 5가 되면서 추가 및 보완된 점

 

 

1. 태그들이 이전에 비해 정형화되었습니다. (구글에서 html5를 검색하시면 위키에 해당 내용이 잘 나옵니다.)

2. 로컬 스토리지 : 오프라인에서도 데이터 저장이 가능합니다. (w3schools 의 web storage 메뉴에서 상세 설명이 잘 돼있습니다.)

3.  미디어와 그래픽 기능이 사용하기 쉬워졌습니다.

video (동영상 플레이 태그), audio (오디오 플레이어 태그), canvas (캔버스처럼 그리기 가능한 태그) - w3schools 의 HTML Media 메뉴 참조

 

4. 위치 정보 API 사용해서 유저의 지리적 위치를 파악할 수 있습니다. -w3schools 의 Geolocation API 메뉴 참조

 

 

 

참조하라고 말씀드리는 사이트들은 좀 더 디테일하게 알고 싶은 분들을 위한 용도입니다.
