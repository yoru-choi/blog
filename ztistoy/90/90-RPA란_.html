RPA란?

 

RPA (Robotic Process Automation) 소프트웨어 로봇을 사용해 업무처리를 자동화하는 기술을 말합니다. 

쉽게 설명하면 컴퓨터에서 하는 작업의 프로세스를 저장하여 필요할 때 재사용하는 기술입니다.

 

RPA 글로벌 시장에서 잘 보이는 UiPath 같은 프로그램이 있습니다.

이 프로그램을 사용한 경우를 예로 들겠습니다.

엑셀의 데이터를 사용해 RPA 프로그램으로 로그인 처리를 진행할 경우

 

[ (URL, ID, PW 가 존재하는) 엑셀 파일 지정 -> 데이터 추출 -> 해당 URL 사이트 오픈 -> ID, PW 입력 ->

로그인 버튼 클릭 -> 로그인 완료 ]      

요런 프로세스를 만들 수 있는데 이 프로세스를 만들고 저장하신다면 다음번엔

클릭 한 번으로 해당 프로세스를 사용할 수 있습니다. 

 

참신한 기술이라기보단 기술이 발전하면서 사람이 할 수 있는 일에 자동화를 외치다 보니

RPA까지 오지 않았나 하는 생각이 듭니다.

간단한 테스트를 통해 RPA가 어떤느낌인지 알고 싶으시다면 링크를 참조해주세요. 감사합니다.
