- 변수의 가독성

변수의 가독성은 주석없이 변수만으로 어떤기능인지 해석될때 가장 좋다고 생각합니다

이때 접두어를 유용하게 사용하면 변수명을 보다 효과적으로 사용할수있습니다

  • is + 단어 : ~ 인지 아닌 boolean 값을 리턴
    e.g) isVaild : 이것이 유효한지?
  • can + 단어 : ~ 가 가능한지 아닌지 boolean 값을 리턴
    e.g) canRead : 읽을 수있는지?(권한)
  • has + 단어 : ~ 가지고 있는지 여부 boolean 값을 리턴
    e.g) hasItem : 그 가이템을 가지고있는지?
  • on + 단어 : ~  boolean 값을 리턴
    e.g) onClick : 버튼을 눌렀는지?

get, put, create, req, num, count, min, max, input, output, temp, dir
prev, curr, next, last, init, def
